# MobiCore Loader Format (v0.0.1)
Author: **vrzh**

_Loader for MobiCore trustlets and drivers_

## Description:

MCLF is a format for MobiCore trustlets and drivers. This plugin will map the trustlet and resolve calls to `mcLib`.
